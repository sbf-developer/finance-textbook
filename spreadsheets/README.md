# Spreadsheet laboratory

`finance_workbook_builder.mjs` creates the companion workbook `finance_quantitative_markets_lab.xlsx`.

The workbook is intentionally transparent: assumptions are separated from calculations, formulas remain visible, source notes are included, and the checks sheet verifies the DCF bridge, portfolio weights and variance, bond price, option value, and formula-error scan.

The workbook's numerical examples use the same fictional Northstar case and simulated-data policy as the textbook. They are teaching inputs, not forecasts or recommendations.
